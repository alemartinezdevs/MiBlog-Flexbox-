HTML

Estructuración del Contenido: HTML define la estructura básica de una página web mediante el uso de elementos como html, head, body, header, footer, nav, entre otros. Estos elementos ayudan a organizar el contenido de la página de manera lógica y jerárquica.

CSS

Selección y Estilo de Elementos: CSS permite seleccionar elementos HTML y aplicar estilos a ellos. Puedes seleccionar elementos específicos, clases o identificadores, y luego definir propiedades como color, tamaño de fuente, márgenes, rellenos, etc.

JavaScript

Interactividad del Usuario: JavaScript permite interactuar con el usuario directamente en la página web. Puedes responder a eventos como clics de botones, movimientos del ratón, ingreso de datos en formularios, entre otros.

Responsive Web

Media Queries: Se utilizan las media queries en CSS para aplicar estilos específicos según las características del dispositivo, como el ancho de la pantalla. Esto permite que el diseño se adapte de manera fluida a diferentes tamaños de pantalla.
